## Overview

A rugged case for the Raspberry Pi 2, 3, 3B/2B/B+ paired with a [2.2" 320×240 resistive TFT display with 6 buttons](https://www.amazon.com/Display-320x240-Resistive-Support-Raspberry/dp/B0982BY7B7).

Running a Raspberry Pi 3 headless on Raspbian Lite is practical for simple dedicated jobs, but sometimes you need just enough display and input to interact with it directly — without dragging out a monitor and keyboard. This case packages the Pi and the TFT display into a self-contained, rugged unit that provides exactly that.

A metal cradle variant is also included, sized for an aluminum equivalent of the TFT module that is no longer for sale.
